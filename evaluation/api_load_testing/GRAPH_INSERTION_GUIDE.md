## 💡 Dicas para Inserção no Word/Google Docs

1. **Localize o texto placeholder:** Procure por `[INSERIR GRÁFICO X]`
2. **Delete o placeholder**
3. **Insira a imagem:** Menu Inserir → Imagem → Escolha o arquivo PNG
4. **Ajuste o tamanho:** Recomendado largura de 15-16cm para o corpo do texto
5. **Adicione a legenda:** Copie o texto da legenda abaixo do placeholder
6. **Formate a legenda:** Texto em itálico, fonte menor (9-10pt)
7. **Centralize:** Tanto a imagem quanto a legenda devem estar centralizadas

## 📞 Troubleshooting

**Problema:** Imagem fica borrada no Word
**Solução:** Use "Inserir como Imagem" não "Copiar/Colar", mantenha a opção "Não comprimir imagens"

**Problema:** Gráfico muito grande/pequeno
**Solução:** Arraste pelos cantos mantendo proporção (Shift pressionado)

**Problema:** Legenda não se alinha
**Solução:** Use estilo de parágrafo "Legenda" ou crie um estilo personalizado

**Problema:** Números de figura incorretos
**Solução:** Use "Inserir → Legenda" no Word para numeração automática
